mofbuilder.core.superimpose.match_vectors(arr1: ndarray, arr2: ndarray, num: int) → Tuple[ndarray, ndarray][source]

Select num points from each set by distance-from-first ordering and return aligned subsets.

Picks the num closest points to the first element in each array, then returns those subsets in the same distance order for use in superimposition.

Parameters:

  • arr1 – First (N1, 3) array of points.

  • arr2 – Second (N2, 3) array of points.

  • num – Number of points to select from each (e.g. min(6, len(arr1), len(arr2))).

Returns:

(num, 3) arrays.

Return type:

Tuple (closest_vectors_arr1, closest_vectors_arr2)

mofbuilder.core.superimpose.superimpose(src_arr: ndarray | List, target_arr: ndarray | List, min_rmsd: float = 1000000.0) → Tuple[float, ndarray, ndarray][source]

Find the best rotation and translation that aligns src_arr to target_arr.

Procedure: - Convert inputs to numpy arrays. - Select up to 6 matching vectors from each set based on distance patterns

(using match_vectors). This reduces the search space for correspondences.

  • Try all permutations of the selected vectors from arr1 and compute the SVD-based superposition against the selected vectors from arr2.

  • Keep the rotation/translation that yields the smallest RMSD.

Parameters:

  • src_arr – Source point set (N, 3).

  • target_arr – Target point set (M, 3).

  • min_rmsd – Initial RMSD threshold; best solution below this is kept.

Returns:

best RMSD, 3x3 rotation matrix, translation vector (length 3).

Return type:

Tuple of (min_rmsd, best_rot, best_tran)

mofbuilder.core.superimpose.svd_superimpose(src_arr: ndarray | List, target_arr: ndarray | List) → Tuple[float, ndarray, ndarray][source]

Compute RMSD and rotation/translation for superimposing two point sets via SVD.

Ref.: “Least-Squares Fitting of Two 3-D Point Sets”, IEEE Trans. Pattern Anal. Mach. Intell., 1987, PAMI-9(5), 698-700. DOI: 10.1109/TPAMI.1987.4767965

Parameters:

  • src_arr – Source point set (N, 3).

  • target_arr – Target point set (M, 3); N should equal M for meaningful RMSD.

Returns:

RMSD, 3x3 rotation matrix, translation vector.

Return type:

Tuple of (rmsd, rot_mat, trans)

mofbuilder.core.superimpose.superimpose_rotation_only(arr1: ndarray | List, arr2: ndarray | List, min_rmsd: float = 1000000.0) → Tuple[float, ndarray, ndarray][source]

Find the best rotation (no translation) that aligns arr1 to arr2 by minimizing RMSD.

Uses the same permutation search over matched subsets as superimpose, but keeps translation fixed (identity). Useful when only orientation matters.

Parameters:

  • arr1 – Source point set (N, 3).

  • arr2 – Target point set (M, 3).

  • min_rmsd – Initial RMSD threshold; best solution below this is kept.

Returns:

best RMSD, 3x3 rotation, translation (often zero).

Return type:

Tuple (min_rmsd, best_rot, best_tran)
