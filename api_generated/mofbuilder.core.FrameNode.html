mofbuilder.core.FrameNode#

class mofbuilder.core.FrameNode(comm=None, ostream=None, filepath=None)[source]#

Bases: object

Load and process metal node PDB: recenter, build graph, optionally add dummy atoms (HHO, HO, O).

Node file should have unique atom names/indices. Produces node_data, nodeG/sG, and (if dummy_node) new PDB and dummy_node_split_dict for use in the builder.
