mofbuilder.core.framework.Framework#

class mofbuilder.core.framework.Framework(comm=None, ostream=None)[source]#

Bases: object

Holds the built MOF: graph, cell, merged data, and options for writing, solvation, and MD.

Set by MetalOrganicFrameworkBuilder.build(). Provides write() for file output, replace() for defects, and access to solvationbuilder and linker FF settings.
