mofbuilder.core.NetOptimizer#

class mofbuilder.core.NetOptimizer(comm=None, ostream=None)[source]#

Bases: object

Optimizes node rotations and cell parameters so linkers fit the net, then places edges.

Uses OptimizationDriver for two-stage rotation optimization and cell scaling. Requires G, V_data, V_X_data, E_data, E_X_data, sorted_nodes, sorted_edges, cell_info, and linker_frag_length (and optionally EC_data for multitopic). Sets opt_rots, sc_unit_cell, sc_rot_node_X_pos, sG, optimized_pair, etc.
