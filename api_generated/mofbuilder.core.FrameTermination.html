mofbuilder.core.FrameTermination#

class mofbuilder.core.FrameTermination(comm=None, ostream=None, filepath=None)[source]#

Bases: object

Loads and holds termination group geometry (e.g. acetate) for capping unsaturated nodes.

Reads a PDB file containing X and Y atom types (e.g. X = connection atom, Y = O-O center), recenters to Y, and exposes termination_data and X/Y subsets.

Parameters:

  • comm (Optional[Any])

  • ostream (Optional[Any])

  • filepath (Optional[str])

Subarray of termination_data for X atoms (set by create).

termination_X_data#

termination_Y_data#

Rows of termination_data where last column equals Y_atom_type (set by create).
