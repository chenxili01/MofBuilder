API Reference#

The API reference below intentionally focuses on public interfaces used in typical workflows. Internal helper functions and imported implementation details are omitted to keep the reference concise.

Builder Workflow#

mofbuilder.MetalOrganicFrameworkBuilder

Orchestrates MOF building: load net and topology, place nodes and linkers, optimize, supercell, defects, write.

mofbuilder.core.framework.Framework

Holds the built MOF: graph, cell, merged data, and options for writing, solvation, and MD.

Core Components#

mofbuilder.core.FrameNet

Build the net graph (G) from a topology CIF: vertices V/CV, edges, cell_info, unit_cell, sorted_nodes/edges.

mofbuilder.core.FrameNode

Load and process metal node PDB: recenter, build graph, optionally add dummy atoms (HHO, HO, O).

mofbuilder.core.FrameLinker

Process organic linkers (xyz/molecule): build graph, find center/outer fragments and X atoms for connection.

mofbuilder.core.FrameTermination

Loads and holds termination group geometry (e.g. acetate) for capping unsaturated nodes.

mofbuilder.core.MofTopLibrary

Lookup and management of MOF families, metals, and topology template CIFs.

mofbuilder.core.NetOptimizer

Optimizes node rotations and cell parameters so linkers fit the net, then places edges.

mofbuilder.core.SupercellBuilder

Build supercell graph from primitive cell graph sG and optionally add virtual edges.

mofbuilder.core.EdgeGraphBuilder

Build edge graph (eG) from supercell graph superG: V/EDGE nodes, XOO attachment, and optional cleaving.

mofbuilder.core.TerminationDefectGenerator

Generate defects by finding unsaturated nodes, removing nodes/linkers, and adding terminations.

mofbuilder.core.MofWriter

Convert edge graph (eG) to per-residue data and write PDB, GRO, XYZ, or CIF.

Input and Output#

mofbuilder.io.CifReader

Class for reading CIF files, extracting symmetry and atomic site information, and providing atom coordinates in the primitive cell.

mofbuilder.io.CifWriter

Writer for crystallographic CIF files (Crystallographic Information File, .cif).

mofbuilder.io.PdbReader

Class for reading and processing atomic coordinates from PDB files.

mofbuilder.io.PdbWriter

Writer for PDB format structure files.

mofbuilder.io.GroReader

Reader for GROMACS .gro coordinate files.

mofbuilder.io.GroWriter

Write atomic structure data to GROMACS .gro coordinate files.

mofbuilder.io.XyzReader

Reader for XYZ molecular coordinate files.

mofbuilder.io.XyzWriter

Writer for XYZ coordinate files.

Modelling and Simulation#

mofbuilder.md.OpenmmSetup

Set up and execute GROMACS/OPENMM simulations for molecular dynamics.

mofbuilder.md.SolvationBuilder

Builder for the creation of solvated molecular structures in a periodic box.

mofbuilder.md.LinkerForceFieldGenerator

Forcefield generation and mapping for linker molecules in MOFs.

mofbuilder.md.GromacsForcefieldMerger

Main class responsible for assembling and merging GROMACS forcefield files for MOF simulations.

mofbuilder.md.ForceFieldMapper

Mapping utility for transferring GROMACS forcefield sections between isomorphic molecules.

Visualization#

mofbuilder.visualization.Viewer

Visualization interface using py3Dmol for MOFbuilder.

Utilities#

mofbuilder.utils.fetch_pdbfile

Finds PDB files in a directory matching keyword filters.

mofbuilder.utils.unit_cell_to_cartesian_matrix

Convert unit cell parameters to a 3x3 Cartesian transformation matrix.

mofbuilder.utils.fractional_to_cartesian

Convert fractional coordinates to Cartesian coordinates.

mofbuilder.utils.cartesian_to_fractional

Convert Cartesian coordinates to fractional coordinates.
